# To-Do Web App

## Project Description
This is a simple web-based to-do list that allows users to add and remove tasks. It demonstrates basic HTML and JavaScript functionality.

## Technologies Used
- HTML
- JavaScript

## Setup Instructions
1. Download the project
2. Open index.html in a web browser

## Usage
- Type a task
- Click "Add"
- Click a task to remove it
